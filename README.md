# Weather Web Application (React + OpenWeather API)

A dynamic Weather Web Application built using **React** for frontend UI. The system allows users to fetch and display **real-time weather data** for any city, with a responsive and interactive user interface.  

---

## Features

- Search weather by city name  
- Display temperature (°C), weather condition, humidity, and wind speed  
- Dynamic background that changes based on weather condition  
- Loading indicator while fetching data  
- Error handling for invalid or non-existent cities  

---

## Technologies Used

- **React.js**  
- **HTML**  
- **CSS**  
- **JavaScript**  
- **REST API** (OpenWeather API)  

---

## Concepts Implemented

- State management using `useState`  
- Controlled form inputs  
- Dynamic rendering using `map()`  
- API integration with `fetch`  
- Conditional rendering  
- Error handling  
- Responsive UI design
