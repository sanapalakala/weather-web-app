# Weather Web App

A React application that fetches real-time weather data from OpenWeather API.

## Features
- Fetch weather by city
- Shows temperature, humidity, wind speed, and weather condition
- Dynamic background based on weather
- Loading indicator
- Keyboard input (press Enter to search)

## Technologies
- React, HTML, CSS, JavaScript, REST API

## Usage
1. Clone the repository:

   git clone https://github.com/sanapalakala/weather-web-app.git

2. Install dependencies:

   npm install

3. Start the app:

   npm start
